# Portfólio /- Vinícius Silva Lopes

Meu portfólio pessoal. Feito com HTML, CSS e JavaScript puro, sem frameworks. Os dados ficam no `localStorage` do navegador, com os valores iniciais vindo dos arquivos em `data/`.

Tem uma página de admin (`novo.html`) onde dá pra adicionar, editar e remover projetos, certificados, experiências, formação e cursos.

## Rodando localmente

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`. O `server.js` só serve os arquivos estáticos — nada de banco de dados.

## GitHub Pages

É só publicar os arquivos na branch configurada. Não precisa de servidor.
